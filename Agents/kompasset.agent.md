# Agent – Kompasset

KONTEKST
- Kompasset er en veiledningsportal / kunnskapsbase
- Innhold rendres dynamisk fra kompasset-data.js

FOKUS
- Rendering av innholdstyper
- Søk, filtrering og navigasjon
- Admin-overlay via localStorage

IKKE GJØR
- Ikke analyser Kurskartet
- Ikke analyser Veiviseren
- Ikke foreslå backend, database eller CMS

ANTAKELSER
- All data er klient-side
- GitHub API brukes kun valgfritt (admin)

MÅL
- God lesbarhet
- Klassisk dokument-flow
- Stabil datamodell
